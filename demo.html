Basic Integration

This page demonstrates the basic integration of the Vet Chatbot SDK. The chatbot widget should appear in the bottom-right corner.

Try asking questions like:

  • "What vaccines does my dog need?"
  • "Tell me about cat nutrition"
  • "I want to book an appointment"

Integration Code

The chatbot is loaded using a single script tag:

<script src="http://localhost:3000/chatbot.js"></script>

Optional Context Configuration

You can optionally pass context to the chatbot:

<script>
window.VetChatbotConfig = {
  userId: "user_123",
  userName: "John Doe",
  petName: "Buddy",
  source: "demo-page"
};
</script>
<script src="http://localhost:3000/chatbot.js"></script>
